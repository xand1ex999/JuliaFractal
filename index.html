Julia Fractal Mathematical Formula

Main formula: Zn+1 = Zn² + C

where:

  • Zn - complex number at the n-th iteration
  • C - complex constant (fractal parameter)
  • Z0 - initial value (pixel coordinate)

Membership condition: A point belongs to the Julia set if the sequence Zn remains bounded (|Zn| ≤ 2) as n → ∞

Animation: Parameter C changes according to the formulas:

Cx = -0.4 + 0.6 × cos(t × 0.3)
Cy = 0.6 + 0.4 × sin(t × 0.7)

where t is time, creating live transformations of the fractal.
